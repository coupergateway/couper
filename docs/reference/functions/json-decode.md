# json_decode

Parses the given [JSON](https://www.json.org) string `encoded` and, if it is valid,
returns the value it represents.

## Syntax

```hcl
various json_decode(encoded string)
```

## Example

```hcl
json_decode("{\"foo\": 1}")
```

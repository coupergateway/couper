# base64_encode

Encodes Base64 string `decoded` as specified in [RFC 4648](https://datatracker.ietf.org/doc/html/rfc4648).

## Syntax

```hcl
string base64_encode(decoded string)
```

## Example

```hcl
base64_encode("foo")
```

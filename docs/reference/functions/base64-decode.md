# base64_decode

Decodes Base64 string `encoded` as specified in [RFC 4648](https://datatracker.ietf.org/doc/html/rfc4648).

## Syntax

```hcl
string base64_decode(encoded string)
```

## Example

```hcl
base64_decode("Zm9v")
```

# url_encode

URL-encodes a given string `s` according to [RFC 3986](https://datatracker.ietf.org/doc/html/rfc3986).

## Syntax

```hcl
string url_encode(s string)
```

## Example

```hcl
url_encode("abc%&,123")
```
